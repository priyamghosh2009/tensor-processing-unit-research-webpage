About

Priyam Ghosh is a software developer and AI hardware enthusiast. He is a graduate of Class XII from Delhi Public School, Paradip Refinery. His early work included graphics applications and file management programs developed in C and C++, which marked the beginning of his interest in software engineering and computer architecture.

In 2023, he developed Fanclub, a social media platform built using PHP, JavaScript, HTML, and CSS that enabled users to communicate and interact in real time.

He later founded Edunova Pro, an educational platform developed using PHP, HTML, CSS, JavaScript, jQuery, AJAX, and SQL. Alongside the web platform, he also developed its Android application using Java. Edunova Pro was focused on facilitating direct communication between students and teachers and simplifying access to assignments, report cards, announcements, and other academic resources.

He subsequently created DEX OS, a private experimental web operating system emphasizing client-side storage and security.

His projects have reflected interests ranging from web technologies and operating systems to computer architecture and artificial intelligence. Currently, he is engaged in research and development of a Tensor Processing Unit (TPU) Instruction Set Architecture, exploring technologies aimed at advancing AI acceleration and high-performance computing.
